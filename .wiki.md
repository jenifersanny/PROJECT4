# Project Summary
The PNG Heritage Garments Store is a full-stack e-commerce application designed to sell traditional Papua New Guinea garments, such as Kolos and Meri blouses. The project utilizes HTML, JavaScript, PHP, and a MySQL database to provide features including user authentication, product catalog browsing, shopping cart functionality, and order management, all while celebrating the rich cultural heritage of Papua New Guinea.

# Project Module Description
The application consists of the following functional modules:
- **User Authentication**: Handles user login, registration, and session management.
- **Product Management**: CRUD operations for products, including categorization and filtering options.
- **Shopping Cart**: Allows users to add, update, and remove items from their cart.
- **Order Management**: Manages user orders and order history.
- **Admin Panel**: Provides administrative functions to manage products and categories.

# File Description Inventory
- **api/auth.php**: Handles user authentication (login, register, logout).
- **api/cart.php**: Manages shopping cart operations.
- **api/categories.php**: Manages product categories.
- **api/products.php**: Handles product operations.
- **assets/css/style.css**: Contains styles for the application.
- **assets/js/main.js**: Contains main JavaScript logic for client-side functionality.
- **cart.html**: Displays the user's shopping cart.
- **config/database.php**: Contains database connection settings.
- **database.sql**: SQL schema for initializing the database.
- **home.html**: Displays the homepage with featured products.
- **includes/functions.php**: Contains core functions for the application.
- **index.html**: The landing page for user authentication.
- **shop.html**: Displays the product catalog.
- **todo.md**: Lists tasks and features to be implemented.
- **uploads/PngMarketplace/**: Directory for uploaded files related to the project.

# Technology Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: PHP
- **Database**: MySQL
- **Frameworks**: None (Vanilla JS and PHP)

# Usage
1. Import `database.sql` into your MySQL database.
2. Update database credentials in `config/database.php`.
3. Place the project files in your web server directory.
4. Access the application via the `index.html` file.
